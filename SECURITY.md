# Security Policy

## Supported Versions

VibeGuard is pre-1.0. Security fixes are released on the latest `0.x` line.

## Reporting A Vulnerability

Do not open public issues for suspected vulnerabilities in VibeGuard itself.

Email: security@example.invalid

Include the affected version, reproduction steps, impact, and whether source code, secrets, scan output, or AI BOM data can be exposed.

## Data Handling

VibeGuard is local-first. It does not upload source code, prompts, AI BOM data, graph data, findings, or secrets by default.
